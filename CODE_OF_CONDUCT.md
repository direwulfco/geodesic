# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its code of conduct.

The full text is published at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be respectful, be constructive, assume good faith, and help others feel welcome contributing. Behavior that makes the community less safe or productive is not tolerated regardless of who it comes from.

## Scope

This Code of Conduct applies in all project spaces — issues, pull requests, discussions, code reviews, the Crystal Store repository, and any other Geodesic-affiliated channel. It also applies when an individual is representing the project in public spaces.

## Reporting

To report a concern, contact the project maintainers at **info@direwulf.com**.

All reports are reviewed promptly, kept confidential, and handled with the goal of making the community better — not punishing the reporter.

## Enforcement

Maintainers are responsible for clarifying and enforcing the standards in the Contributor Covenant linked above. They may take any action they deem appropriate in response to a report, ranging from a private clarification to removal from project spaces.
